Full Stack Developers

Full Stack Developers :

Full Stack Developers are categorized into two types. They are :

  1. Frontend Developers

  2. Backend Developers

In Frontend, we create user interference with a programming languages like HTML, CSS, JavaScript, etc...


In Backend, we use it for storing and organizing data, and ensuring everything on the client side actually works.


Similarly, we use DataBase in the Backend itself !

Examples of DataBases like :

  1. My SQL. [ Examples : MySQl is a relational database management system.]

  2. Microsoft Access. [ Examples : Flat file database and relational databases.]

  3. dBASE. [ dBASE is used for retrieving data in various formats, primarily for database applications.]
